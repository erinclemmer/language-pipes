## Supported model families

- **Qwen3** (dense)
- **Qwen3-MoE** (mixture-of-experts)
- **Phi** (Phi-3 / Phi-4 architecture)
- **Meta Llama** (Llama 2 and Llama 3.x)
- **Gemma 3**
- **Gemma 4**
- **Ministral 3**

Note: Any fine-tuned model based on a supported base architecture should also work.
